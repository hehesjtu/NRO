# NRO: Remote Sensing Image Detail Enhancement Based on Newton Raphson Optimization

This is the source code of paper named "Remote Sensing Image Detail Enhancement Based on Newton Raphson Optimization". This paper is submitted to IEEE TRANSACTIONS ON GEOSCIENCE AND REMOTE SENSING.

## Introduction

This repository contains the MATLAB implementation of the NRO algorithm proposed in the paper:

"[Remote Sensing Image Detail Enhancement Based on Newton Raphson Optimization]"

Authors:[He Jiang {Member, IEEE}], [Hengshuo Liu], [Luran Chen]

**Abstract:** Residual learning-based methods for remote sens-
ing image detail enhancement have demonstrated promising
performance. However, they typically rely on greedy search
strategy, which are prone to getting trapped in local optima
and thereby constrain the model performance. In contrast, the
Newton Raphson root finding method leverages second order
derivative information to guide the optimization process, effec-
tively mitigating the tendency to converge prematurely to sub-
optimal solutions.  Inspired by this property, this study proposes
aNewton Raphson Optimization based remote sensing image
detail enhancement method, i.e., NRO.

Firstly, NRO adopts
an interpolation-based strategy for detail feature extraction to
decouple coarse texture features. Secondly, NRO generates an
adaptive texture mask guided by local texture content. Thirdly,
the optimization framework integrates three core components,
namely a nonlinear and center-biased warm start initialization,
the Newton Raphson Search Rule (NRSR), and a gradient-
guided Trap Avoidance Operator (TAO), jointly improving model
accuracy and ensuring global convergence.

**Results:** Extensive experiments
demonstrate consistent performance gains of NRO across three
benchmark datasets(WHU-RS19, SIRI-WHU, UCMerced). On the WHU-RS19 dataset with a ×4
enhancement factor, our NRO outperforms the popular PMN 
in detail enhancement, achieving a 9.28 dB improvement in PSNR
and a 0.1361 increase in SSIM. These results further validate
NRO’s superiority in remote sensing image detail enhancement
and its strong generalization capability across diverse scene types.

## System Requirements

The code has been tested and verified on the following platform:

* **MATLAB Version:** R2023a or later
* **Operating System:** Windows 10/11

### Required Toolboxes
To run the code successfully, the following MATLAB toolboxes are required:

* **Image Processing Toolbox** (Required for image I/O, resizing, and metric calculation)
* **Parallel Computing Toolbox** (Required for the `parpool` acceleration used in the core algorithm)

## Project Structure
The following directory hierarchy outlines the organization of the repository:
```
NRO/
├── data/                   # Sample images for testing purposes
│   ├── example1.png
│   └── example2.png
│   └── ...
├── results/                # Enhanced results generated by NRO
│   ├── example1_enhanced.png
│   └── example2_enhanced.png
│   └── ...
├── NRO.p                   # Pre-compiled core algorithm (P-code)
├── demo.m                  # Main entry script with GUI
├── README.md               # Project documentation
├── nro_solver.mexw64       # Pre-compiled C++ core solver for Windows
```
This implementation provides a user-friendly GUI for testing the algorithm on custom images.

1.  Open MATLAB and navigate to the repository folder.
2.  In the MATLAB Command Window, type the following command to launch the interface:
    ```matlab
       demo
    ```
3.  GUI Operations:
    Click **"Load Image"**  to select an input image (supports .png, .jpg, .bmp, .tif).
     Adjust the **"Enhance Factor"** if necessary (Default is 4.0).
     Click **"Launch NRO"** to process the image. The status bar will indicate the progress.
     Upon completion, the PSNR and SSIM metrics will be displayed in the interface.
     Click **"Save Result"**  to export the enhanced image.
    
## File Description
 `demo.m`: The main entry point script containing the GUI implementation and interaction logic.

 `NRO.p`: The pre-compiled binary file of the core NRO algorithm. 
  Note: The core algorithm is provided as a P-code file for copyright protection. It is functionally identical to the source code described in the paper.
  
 `data/`: (Optional) Contains sample images for testing.

 `results/`: Stores the corresponding enhanced images processed by the NRO algorithm, serving as a quality reference.

 `nro_solver.mexw64`: The Pre-compiled C++ core solver for Windows
 Note:The code is provided as a mexw64 file for copyright protection. It is functionally identical to the source code described in the paper.
